Proteção dos seus dados pessoais

Conformidade LGPD

Lei Geral de Proteção de Dados

O tratamento de dados pessoais de fornecedores (Nome, E-mail, Telefone, CPF/CNPJ) neste sistema tem como base legal a Execução de Contrato e o Legítimo Interesse (Art. 7º da Lei 13.709/2018), sendo indispensável para a homologação e pagamento dos serviços prestados.

Finalidade e Segurança

Uso restrito das informações

Os dados coletados (incluindo comprovantes visuais) são utilizados exclusivamente para fins de auditoria de mídia ("Comprovação de Veiculação"). Adotamos medidas técnicas rigorosas para proteger essas informações contra acessos não autorizados.
